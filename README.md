# Bean Scene Reservation ASP.NET Web Application

![Bean Scene Logo](Bean-Scene-Reservation\wwwroot\assets\images\logo\png\logo-secondary.png)

A restaurant reservation applications which allow staff and customers to reserve tables for scheduled sittings.

Made for the fictional restaurant Bean Scene.

## Technology Involved

- ASP.NET (Active Server Pages for .NET)
- MVC (Model-View-Controller)
- EF (Entity Framework)
